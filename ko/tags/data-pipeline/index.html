신호 운영용 Headway/GAP KPI 파이프라인

2026. 4. 28.

목표 차간간격(Headway/GAP) 기반의 교통류 KPI(메타데이터) 를 안정적으로 산출해 공용 데이터베이스에 적재하고, 신호 회사가 이를 신호 운영 개선 지표로 활용할 수 있게 만드는 것이 주 목적입니다. 구현 내용 차로 단위 이벤트 처리: 차로별 그룹화 → 시간순 정렬 → 연속 차량 쌍 생성 현시(Phase) 기반 유효성: 동일 현시 구간 내 연속쌍만 채택(현시 경계를 넘는 혼합 방지) KPI 계산: Headway = Δt (초) GAP = Headway - (차량길이 / 속도(m/s)) (단위 변환 포함) 품질 규칙: 너무 짧거나 긴 간격, 물리적으로 불합리한 값 등 노이즈/이상치 제거 후 사이클 단위로 평균/합계 집계(임계값 비공개) 메타데이터 적재: 사이클별 KPI(평균/합계, 유효 물량, 초기/후반 구간 집계 등)를 공용 RDB(Tibero) 에 적재해 신호 회사의 리포팅/운영에 활용 가능하게 구성 어려웠던 점(카메라 기반 검지) 상위 데이터는 레이더가 아니라 카메라 이미지 디텍션 + ROI(통과 영역) 기반이어서, 교차로마다 화각/설치 각도 차이와 가림/근접 주행 등의 영향으로 트래킹 품질이 흔들릴 때가 있었습니다(예: 차량이 너무 붙어 ID가 섞이거나, ROI 통과 이벤트가 누락/중복되는 케이스).
